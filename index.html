Nous sommes en train de développer une fonction de raccourcis clavier et de configuration du contrôleur, dont une partie sera mise en ligne dans la version 1.1. Nous tirons sur tous les cylindres pour compléter les fonctionnalités restantes, qui deviendront disponibles pour les joueurs dans une version ultérieure.

J'aurais aimé qu'il y ait plus de façons de filtrer les artefacts.

Dans la version 1.1, nous avons ajouté de nouveaux filtres pour les artefacts basés sur leurs attributs de base. Désormais, les voyageurs pourront facilement filtrer les artefacts par attributs.

Traverser le monde du jeu prend du temps, je veux pouvoir définir mes propres waypoints de téléportation pour les zones que je visite fréquemment.

Nous avons développé un élément qui fournit ce type de fonction qui sera disponible gratuitement dans un nouveau système. Les voyageurs peuvent le rechercher dans la version 1.1!

Comment le DMG est-il calculé lorsque j'ai plus d'un bouclier? Et quel bouclier soutient DMG en premier?

En pratique, lorsqu'un personnage est touché, le DMG soutenu affectera simultanément chaque bouclier actif. Une fois ce DMG calculé, si le personnage a encore des boucliers, le personnage lui-même ne prendra pas de DMG.
